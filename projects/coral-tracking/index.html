This is a team project, requested by my lecturer. The idea of this app to do coral detection and classification to track the distribution of coral reefs in Indonesia. There is four people in this team, I have responsibility to create the machine learning model to detect and identification the coral inputted by mobile devices. The model is created using python and keras model. This model got 90% accuracy to detect and identification coral. I also have responsibility to create mobile app with flutter to capture coral image and upload to the server and serve the result back to the user.
